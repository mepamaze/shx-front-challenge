# FRONT-END SHX Admission Test

## Dependencies
- Node 12.14.1
- Angular 12.2.x
- VSCode Extension: Angular Language Service 12.2.x !IMPORTANT

## How to Install
```bash
npm install
```

## How to Run
```bash
npm run start
```
